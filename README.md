# User Profile Form

A simple SwiftUI application that allows users to enter their name and email address and display the submitted information on the screen.

This project was built as part of SwiftUI learning and demonstrates the fundamentals of Swift programming, state management, user input handling, and UI design.

---

## 📱 Demo

The application provides:

* User Name input field
* Email input field
* Submit button
* Display section showing submitted user information

After entering the details and tapping **Submit**, the entered information is displayed below the form.

---

## ✨ Features

* User input using `TextField`

## 🚀 How It Works

1. User enters Name and Email.
2. Tap the Submit button.
3. The entered values are stored in state variables.
4. SwiftUI automatically refreshes the UI.
5. Submitted information appears in the User Information section.
